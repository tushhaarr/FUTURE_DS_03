Marketing Funnel & Conversion Analysis

Project Overview:
This project analyzes a bank telemarketing campaign dataset to identify funnel drop-offs and high-converting customer segments using Power BI.
The goal was to uncover actionable insights to improve lead-to-customer conversion rates.

Business Objective:
- Analyze marketing funnel performance
- Measure conversion rate
- Identify high-value customer segments
- Provide strategic recommendations

Tools Used:
- Power BI
- DAX
- Power Query
- Data Modeling

Key KPIs:
- Total Contacts
- Total Subscriptions
- Conversion Rate
- Segment-Level Conversion

Key Insights:
- Overall conversion rate ~12%
- Previous successful campaign customers convert significantly higher
- High balance customers show stronger conversion rates
- Certain job categories outperform others

Recommendations:
- Retarget previous successful customers
- Focus campaigns on high-balance segments
- Prioritize effective contact channels
- Personalize marketing for high-performing demographics
